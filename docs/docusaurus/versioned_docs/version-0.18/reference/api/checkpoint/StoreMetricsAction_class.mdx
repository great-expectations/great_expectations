<dd>
<p>Extract metrics from a Validation Result and store them in a metrics store.</p>
<p>Typical usage example:</p>
<CodeBlock language="yaml">{`    - name: store_evaluation_params
    action:
     class_name: StoreMetricsAction
      # the name must refer to a store that is configured in the great_expectations.yml file
      target_store_name: my_metrics_store`}</CodeBlock>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>data_context</strong> – GX Data Context.</p></li>
<li><p><strong>requested_metrics</strong> – <p>Dictionary of metrics to store.</p></p></li>
<p>Dictionary should have the following structure:</p>
</ul>
</dd>
</dl>
<dl class="simple">
<dt><CodeBlock language="yaml">{`                                  expectation_suite_name:
                                      metric_name:
                                          - metric_kwargs_id`}</CodeBlock></dt>
<dd>
<p>You may use "*" to denote that any expectation suite should match.</p>
</dd>
</dl>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd">
<p><strong>target_store_name</strong> – The name of the store where the action will store the metrics.</p>
</dd>
<dt class="field-even">Raises</dt>
<dd class="field-even"><ul class="simple">
<li><p><strong>DataContextError</strong> – Unable to find store {} in your DataContext configuration.</p></li>
<li><p><strong>DataContextError</strong> – StoreMetricsAction must have a valid MetricsStore for its target store.</p></li>
<li><p><strong>TypeError</strong> – validation_result_id must be of type ValidationResultIdentifier or GeCloudIdentifier, not {}.</p></li>
</ul>
</dd>
</dl>

<p>Public entrypoint GX uses to trigger a ValidationAction.</p>
<p>When a ValidationAction is configured in a Checkpoint, this method gets called
after the Checkpoint produces an ExpectationSuiteValidationResult.</p>

<li><p><strong>data_asset</strong> – An instance of the Validator class.</p></li>
<li><p><strong>expectation_suite_identifier</strong> – Optionally, an instance of the ExpectationSuiteIdentifier class.</p></li>
