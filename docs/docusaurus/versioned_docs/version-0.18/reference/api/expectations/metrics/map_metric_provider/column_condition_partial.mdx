<p>Provides engine-specific support for authoring a metric_fn with a simplified signature.</p>
<p>A column_condition_partial must provide a map function that evaluates to a boolean value; it will be used to provide
supplemental metrics, such as the unexpected_value count, unexpected_values, and unexpected_rows.</p>
<p>A metric function that is decorated as a column_condition_partial will be called with the engine-specific column
type and any value_kwargs associated with the Metric for which the provider function is being declared.</p>
