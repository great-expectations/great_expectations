<p>Base class for all Query Metrics, which define metrics to construct SQL queries.</p>
<blockquote>
<div>
<p>An example of this is <cite>query.table</cite>, which takes in a SQL query &amp; target table name, and returns the result of that query.</p>
<p>In some cases, subclasses of MetricProvider, such as QueryMetricProvider, will alreadyhave correct values that may simply be inherited by Metric classes.</p>
