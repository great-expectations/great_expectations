<dd>
<p>Defines the parameters and name of a specific Expectation.</p>
<dl class="field-list simple">
<dt class="field-odd">Parameters
</dt>

<dd class="field-odd"><ul class="simple">
<li><p><strong>expectation_type</strong> – The name of the expectation class to use in snake case, e.g. <cite>expect_column_values_to_not_be_null</cite>.</p></li>
<li><p><strong>kwargs</strong> – The keyword arguments to pass to the expectation class.</p></li>
<li><p><strong>meta</strong> – A dictionary of metadata to attach to the expectation.</p></li>
<li><p><strong>success_on_last_run</strong> – Whether the expectation succeeded on the last run.</p></li>
<li><p><strong>ge_cloud_id</strong> – <p>The corresponding GX Cloud ID for the expectation.</p>
<div class="versionadded">
<p><span class="versionmodified added">New in version 0.13.36: </span>Used in GX Cloud deployments.</p>
</div>
</p></li>
<li><p><strong>expectation_context</strong> – <p>The context for the expectation.</p>
<div class="versionadded">
<p><span class="versionmodified added">New in version 0.13.44: </span>Used to support column descriptions in GX Cloud.</p>
</div>
</p></li>
<li><p><strong>rendered_content</strong> – <p>Rendered content for the expectation.</p>
<div class="versionadded">
<p><span class="versionmodified added">New in version 0.15.14: </span>Used to include rendered content dictionary in expectation configuration.</p>
</div>
</p></li>
</ul>
</dd>
<dt class="field-even">Raises
</dt>

<dd class="field-even"><ul class="simple">
<li><p><strong>InvalidExpectationConfigurationError</strong> – If <cite>expectation_type</cite> arg is not a str.</p></li>
<li><p><strong>InvalidExpectationConfigurationError</strong> – If <cite>kwargs</cite> arg is not a dict.</p></li>
<li><p><strong>InvalidExpectationKwargsError</strong> – If domain kwargs are missing.</p></li>
<li><p><strong>ValueError</strong> – If a <cite>domain_type</cite> cannot be determined.</p></li>
</ul>
</dd>
