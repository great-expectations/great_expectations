<p>Base class for all RegexColumnMapMetrics.</p>
<p>RegexColumnMapMetric classes inheriting from RegexColumnMapMetricProvider are ephemeral,
defined by their <cite>regex</cite> attribute, and registered during the execution of their associated RegexColumnMapExpectation.</p>
<p>Metric Registration Example:</p>
<p><CodeBlock language="python">{`map_metric = RegexBasedColumnMapExpectation.register_metric(
    regex_camel_name='Vowel',
    regex_='^[aeiouyAEIOUY]*$',
)`}</CodeBlock></p>
<p>In some cases, subclasses of MetricProvider, such as RegexColumnMapMetricProvider, will already
have correct values that may simply be inherited by Metric classes.</p>
