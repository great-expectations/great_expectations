<dd>
<p>Suite of expectations plus create, read, update, and delete functionality.</p>
<ul class="simple">
<li><p>create: add_expectation(), append_expectation()</p></li>
<li><p>read: find_expectation_indexes(), find_expectations(), show_expectations_by_domain_type(), show_expectations_by_expectation_type()</p></li>
<li><p>update: add_expectation(), append_expectation(), patch_expectation(), replace_expectation(), add_expectation_configurations()</p></li>
<li><p>delete: remove_expectation(), remove_all_expectations_of_type()</p></li>
</ul>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>expectation_suite_name</strong> – Name of the Expectation Suite.</p></li>
<li><p><strong>data_context</strong> – Data Context associated with this Expectation Suite.</p></li>
<li><p><strong>expectations</strong> – Expectation Configurations to associate with this Expectation Suite.</p></li>
<li><p><strong>evaluation_parameters</strong> – Evaluation parameters to be substituted when evaluating Expectations.</p></li>
<li><p><strong>data_asset_type</strong> – Type of data asset to associate with this suite.</p></li>
<div class="deprecated">
<p><span class="versionmodified deprecated">Deprecated since version 0.14.0.</span></p>
</div>
<li><p><strong>execution_engine_type</strong> – Name of the execution engine type.</p></li>
<li><p><strong>meta</strong> – Metadata related to the suite.</p></li>
<li><p><strong>ge_cloud_id</strong> – Great Expectations Cloud id for this Expectation Suite.</p></li>
<div class="versionadded">
<p><span class="versionmodified added">New in version 0.13.33: </span>Used in GX Cloud deployments.</p>
</div>
</ul>
</dd>
