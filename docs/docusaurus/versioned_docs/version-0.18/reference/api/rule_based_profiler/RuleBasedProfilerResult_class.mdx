<p><code class="docutils literal notranslate"><span class="pre">RuleBasedProfilerResult</span></code> is an immutable <code class="docutils literal notranslate"><span class="pre">dataclass</span></code> object which holds the results of executing the <code class="docutils literal notranslate"><span class="pre">RuleBasedProfiler.run()</span></code> method.</p>
<p>Properties represents the configuration of the Rule-Based Profiler (effective configuration if run via a DataAssistant or auto-initializing expectation), with all run-time overrides properly reconciled.</p>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>fully_qualified_parameter_names_by_domain</strong> – <cite>dict</cite> of <cite>Domain</cite> keys and a list of their parameter names.</p></li>
<li><p><strong>parameter_values_for_fully_qualified_parameter_names_by_domain</strong> – <cite>dict</cite> of <cite>Domain</cite> and nested <cite>ParameterNode</cite> mappings.</p></li>
<li><p><strong>expectation_configurations</strong> – List of <cite>ExpectationConfiguration</cite> objects.</p></li>
<li><p><strong>citation</strong> – <cite>dict</cite> of citations.</p></li>
<li><p><strong>catch_exceptions</strong> (<em>boolean</em>) – Defaults to False.
If True, then catch exceptions and include them as part of the result object.             For more detail, see [catch_exceptions](<a class="reference external" href="https://docs.greatexpectations.io/docs/0.18/reference/learn/expectations/standard_arguments/#catch_exceptions">Standard Arguments</a>).</p></li>
</ul>
