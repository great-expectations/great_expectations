<p>Extension of ConfiguredAssetFilePathDataConnector used to connect to GCS.</p>
<p>A ConfiguredAssetGCSDataConnector requires an explicit specification of each DataAsset you want to connect to.
This allows more fine-tuning, but also requires more setup. Please note that in order to maintain consistency
with Google’s official SDK, we utilize terms like “bucket_or_name” and “max_results”. Since we convert these keys from YAML
to Python and directly pass them in to the GCS connection object, maintaining consistency is necessary for proper usage.</p>
<dl class="simple">
<dt>This DataConnector supports the following methods of authentication:</dt>
<dd><ol class="arabic simple">
<li><p>Standard gcloud auth / GOOGLE_APPLICATION_CREDENTIALS environment variable workflow</p></li>
<li><p>Manual creation of credentials from google.oauth2.service_account.Credentials.from_service_account_file</p></li>
<li><p>Manual creation of credentials from google.oauth2.service_account.Credentials.from_service_account_info</p></li>
</ol>
</dd>
</dl>
