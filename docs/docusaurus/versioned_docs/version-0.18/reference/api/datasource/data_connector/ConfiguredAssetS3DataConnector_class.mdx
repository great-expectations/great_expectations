<dd>
<p>Extension of ConfiguredAssetFilePathDataConnector used to connect to S3.</p>
<p>Being a Configured Asset Data Connector, it requires an explicit list of each Data Asset it can
connect to. While this allows for fine-grained control over which Data Assets may be accessed,
it requires more setup.</p>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>name</strong> (<em>str</em>) – required name for DataConnector</p></li>
<li><p><strong>datasource_name</strong> (<em>str</em>) – required name for datasource</p></li>
<li><p><strong>bucket</strong> (<em>str</em>) – bucket for S3</p></li>
<li><p><strong>assets</strong> (<em>dict</em>) – dict of asset configuration (required for ConfiguredAssetDataConnector)</p></li>
<li><p><strong>execution_engine</strong> (<a class="reference internal" href="/docs/0.18/reference/api/execution_engine/ExecutionEngine_class" title="great_expectations.execution_engine.ExecutionEngine"><em>ExecutionEngine</em></a>) – optional reference to ExecutionEngine</p></li>
<li><p><strong>default_regex</strong> (<em>dict</em>) – optional regex configuration for filtering data_references</p></li>
<li><p><strong>sorters</strong> (<em>list</em>) – optional list of sorters for sorting data_references</p></li>
<li><p><strong>prefix</strong> (<em>str</em>) – S3 prefix</p></li>
<li><p><strong>delimiter</strong> (<em>str</em>) – S3 delimiter</p></li>
<li><p><strong>max_keys</strong> (<em>int</em>) – S3 max_keys (default is 1000)</p></li>
<li><p><strong>boto3_options</strong> (<em>dict</em>) – optional boto3 options</p></li>
<li><p><strong>batch_spec_passthrough</strong> (<em>dict</em>) – dictionary with keys that will be added directly to batch_spec</p></li>
