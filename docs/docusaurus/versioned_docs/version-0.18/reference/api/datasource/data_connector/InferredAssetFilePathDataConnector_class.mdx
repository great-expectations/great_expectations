<dd>
<p>A base class for Inferred Asset Data Connectors designed to operate on file paths and implicitly determine Data Asset names through regular expressions.</p>
<p>Note that <cite>InferredAssetFilePathDataConnector</cite> is not meant to be used on its own, but extended.</p>
<dl class="field-list simple">
<dt class="field-odd">Parameters</dt>
<dd class="field-odd"><ul class="simple">
<li><p><strong>name</strong> – The name of the Data Connector.</p></li>
<li><p><strong>datasource_name</strong> – The name of this Data Connector’s Datasource.</p></li>
<li><p><strong>execution_engine</strong> – The Execution Engine object to used by this Data Connector to read the data.</p></li>
<li><p><strong>default_regex</strong> – A regex configuration for filtering data references. The dict can include a regex <cite>pattern</cite> and
a list of <cite>group_names</cite> for capture groups.</p></li>
<li><p><strong>sorters</strong> – A list of sorters for sorting data references.</p></li>
<li><p><strong>batch_spec_passthrough</strong> – Dictionary with keys that will be added directly to the batch spec.</p></li>
<li><p><strong>id</strong> – The unique identifier for this Data Connector used when running in cloud mode.</p></li>
</ul>
</dd>
